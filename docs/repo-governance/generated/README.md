# Generated Repo-Governance Reports

Status: Generated reports folder.

## Purpose

This folder stores generated repository-governance reports.

Generated reports may support cleanup review, inventory review, and repository organization decisions.

## Rules

Generated reports do not move files.

Generated reports do not delete files.

Generated reports do not change source code.

Generated reports do not change tests.

Generated reports do not change workflows.

Generated reports require human review before any repository action.
