# Replaced Documents Archive

Status: Post-MVP archive subfolder.

## Purpose

This folder is reserved for documents that were replaced or superseded by clearer later documents but should remain preserved for decision history.

## Possible Future Contents

This folder may later contain:

- replaced repository maps;
- older status notes;
- superseded planning documents;
- old workflow notes;
- replaced architecture drafts;
- earlier versions of documents now represented by closure reviews or exit audits.

## What Does Not Belong Here

This folder should not contain:

- current MVP proof documents;
- current governance documents;
- active README files;
- active source files;
- active tests;
- workflow files.

## Movement Rule

No files should be moved here until replacement status has been explicitly confirmed.

## Interpretation

Replaced does not mean useless.

Replaced means a newer document is now authoritative, while the old document may still preserve useful context.
