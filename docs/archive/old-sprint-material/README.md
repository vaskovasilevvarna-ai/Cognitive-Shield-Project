# Old Sprint Material Archive

Status: Post-MVP archive subfolder.

## Purpose

This folder is reserved for sprint-related documents that remain historically useful but may no longer need to stay in the active sprint documentation path.

## Possible Future Contents

This folder may later contain:

- intermediate pass notes;
- repeated checkpoint notes;
- old sprint working documents;
- superseded sprint planning files;
- detailed closure fragments replaced by exit audits;
- historical sprint documents preserved for traceability.

## What Does Not Belong Here

This folder should not contain:

- active Sprint 4 MVP proof records;
- current MVP status notes;
- current post-merge verification records;
- current repo-governance cleanup documents;
- current source or test files.

## Movement Rule

No sprint documents should be moved here until a separate archive movement boundary review identifies specific candidate groups.

## Interpretation

Old sprint material may remain valuable.

Moving a document here should reduce active navigation noise while preserving the project reasoning trail.
