# Legacy Planning Archive

Status: Post-MVP archive subfolder.

## Purpose

This folder is reserved for older planning material that should be preserved for historical context but should not clutter the active project navigation path.

## Possible Future Contents

This folder may later contain:

- old roadmap drafts;
- early estimates;
- preliminary implementation plans;
- historical strategy notes;
- older planning documents superseded by post-MVP direction;
- documents that help explain past planning assumptions.

## What Does Not Belong Here

This folder should not contain:

- active post-MVP plans;
- current Functional Local Prototype Engine planning documents;
- active cleanup control documents;
- current MVP proof documents;
- source or test files.

## Movement Rule

No planning documents should be moved here until a separate archive movement review confirms they are historical planning material.

## Interpretation

Legacy planning may remain useful for understanding project evolution.

It should be preserved when it has context value, but separated from current active planning.
